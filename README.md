This is a bike search app that will display the picture of the bike you select
Will have get post put and delete

FUNCTIONALITY
Will have a header saying name of app
drop down boxes will appear
when make model and size and year are selected will show what you 
when hitting find it will display a picture of what you selected.

ENDPOINTS
Get- will get the objects for data I create
Post- will push up the objects to the drop down menu
Put- will allow change in the menu
delete- will allow to delete currently selected options

COMPONENT ARCHITECTURE
App.js (stateful: will have the hardcoded date, this.state.data)
header.js will be the name of app
finder.js will find the certain data using axios
bike.js will find the certain year
